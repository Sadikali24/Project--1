    now we will discuss about types of applications

    1. standalone application
      • The application which does't require internet is refferred as stand alone application
      • To develop stand alone application make use of "core java"
      • This application needs to install in clients machine
      • To run this application user require operating system like windows,linux,unix,mac etc....
      • For example ms-office etc
    2. web application
      • The application which is working using internet and browser is refferred as web application
      • To develop web application make use of "web technologies" (or) "advance java"
      • Web application does't require installation in client machine
      • For example W3 schools,flipkart etc.....

      3. server

      • Server is a mediater between programmer and user
      • It is a machine which is used to store web application
      • User can access web application by sending a request to the server with the help of internet and server will provide response to user
      • Server and client machine present at different locations.Hence,user requires internet for web application

      now we will discuss about components of application

      Web applications have two important components
      1. Frontend component
        • The components which are visible to the user is refferred as "Frontend component "
        • To develop frontend components make use of "Web technologies"
        • For example login page,advertisment etc.....
      2. Backend component
        • The components which are hidden from the user is refferred as "backend component"
        • To develop backend components make use of "core java " and "sql"
        • For example database

      Types of application

      We have two types of web applications
      1. Static web application
      2. Dynamic web application
      1. Static web application
        • The application which contains only frontend components is refferred as "static web application"
        • To develop Static web applications programmers are using "web application"
        • For example blogs..
      2. Dynamic web application
        • The application which contains frontend as well as backend is refferred as "dynamic web application"
        • To develop this type of applications,programmers are using "core java+sql+web technologies(advance java)"
        • For example flipkart,amazon etc.......
    3. mobile application
      • The application which needs to download and install in mobile phone is refferred as mobile application
      • To develop mobile application make use of "flutter technology"
      • To run mobile application user required android (or) ios operating system
      • For example township,pubg etc.......
    4. distributed application
      • The application which are interconnected to each other is refferred as distributed application
      • To develop this application make use of "cloud technology"
      • For example G-mail,G-photos,G-drive etc......
